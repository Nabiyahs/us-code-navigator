## Interaction Flow Diagram

```
User Opens Page
    │
    ├─→ Clicks "Home"
    │       └─→ Shows Advanced Search + Featured Codes
    │
    ├─→ Clicks "코드"
    │       └─→ Dropdown appears
    │               └─→ Clicks code (e.g., "IBC")
    │                       └─→ Shows code content with sections
    │
    ├─→ Clicks "검색"
    │       └─→ Shows search page
    │               └─→ Enters filters
    │                       └─→ Clicks "Search"
    │                               └─→ Shows results
    │                                       └─→ Clicks result
    │                                               └─→ Navigates to code section
    │
    ├─→ Clicks "비교"
    │       └─→ Shows compare page
    │               └─→ Selects two codes
    │                       └─→ Clicks "Compare"
    │                               └─→ Shows side-by-side with diff
    │
    └─→ Uses header search
            └─→ Presses Enter
                    └─→ Navigates to search page with results
```

## State Management

```
Global State Variables:
├── currentPage: 'home' | 'code' | 'search' | 'compare'
└── currentCode: null | 'ibc' | 'nfpa13' | 'nfpa20' | 'nfpa72'

Page States:
├── Home: Always visible when currentPage = 'home'
├── Code: Visible when currentPage = 'code', shows currentCode data
├── Search: Visible when currentPage = 'search', shows search results
└── Compare: Visible when currentPage = 'compare', shows comparison

Navigation States:
├── Active Nav Item: Updated by currentPage
├── Dropdown Open/Closed: Toggled by user interaction
└── Active Dropdown Item: Updated by currentCode
```
